# Scripts de Automação Linux

Automação de criação de usuários, grupos, diretórios e especificação de permissões em sistemas Linux para curso de Linux da DIO.me.

## 📋 Conteúdo

- `criar_usues.sh` – Script para criar usuários do sistema com grupos e permissões
- `iac1.sh` – Script para criar estrutura de diretórios e aplicar permissões

## 🚀 Como Usar
```bash
sudo bash criar_uses.sh
sudo bash iac1.sh
